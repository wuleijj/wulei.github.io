使用Hexo搭建个人网站

发表于 2018-07-06 | 分类于 Hexo

一、博客的整体配置

博客的整体配置在D:\hexo\_config.yml文件中进行。默认使用的主题是landscape，主题的配置在D:\hexo\themes\landscape\_config.yml。主题的配置可以暂时不动，等到更换了自己满意的主题后再进行。先来说一下博客的整体配置，也就是打开D:\hexo\_config.yml文件，更改里面的部分内容。简化起见，其他内容先不做更改，只更改# site下面的内容。

本地预览hexo s -p 9000

二、更换主题

1.新建文章

新建文章使用hexo n命令。也可以在博客目录source\_posts中新建一个后缀为.md的文件，使用vscode等编辑器用Markdown书写。抬头信息只需在文章开头添加：

即可。

2.文章图片存放

想要在文章中插入图片的话，可用Markdown语法插入，格式为![图片名称](地址)。图片的存放有两种，一种在本地source\目录新建存放图片的文件夹，比如image，然后把插入的图片放里面，插入图片的路径；或者把图片上传网络，然后插入图片路径。推荐七牛云存储
